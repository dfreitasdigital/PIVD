## ✨ Funcionalidades

- ✅ Dashboard com indicadores
- ✅ Gestão de pagamentos e fornecedores
- ✅ RH com folha de pagamento
- ✅ Produção assistencial
- ✅ Extrato bancário
- ✅ Balancete contábil
- ✅ Ofícios expedidos
- ✅ **Histórico de versões** dos anexos (IndexedDB)
- ✅ Exportação de backup em ZIP
- ✅ Geração de livro de prestação de contas (PDF)

## 🔧 Tecnologias

- HTML5 / CSS3 / JavaScript
- IndexedDB (armazenamento local persistente)
- jsPDF (geração de PDF)
- JSZip (exportação de backup)
- Font Awesome Icons

## 📱 Acesse pelo celular

O sistema é totalmente responsivo e funciona no celular!

## 📂 Armazenamento

- Todos os anexos são salvos localmente no navegador (IndexedDB)
- Os arquivos permanecem mesmo após fechar o navegador
- Backup completo disponível na aba "📜 Histórico"
